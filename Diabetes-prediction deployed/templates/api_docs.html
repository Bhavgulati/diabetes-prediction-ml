DiabetesAI API Docs

Integrate diabetes risk prediction into any application. Send patient biomarkers and receive instant risk scores powered by our Random Forest model.

Base URL: https://diabetes-prediction-ml-2-dt3u.onrender.com

Endpoints

GET /api/health
Check if the API is running. Use this to verify connectivity before making prediction requests.
Response 200
{ "status": "healthy", "service": "DiabetesAI API", "version": "1.0.0", "timestamp": "2026-03-22T10:00:00" }
POST /api/predict
Submit patient biomarkers and receive a diabetes risk score, category, and confidence level. No authentication required.
Request Body (JSON)
ParameterTypeRange
pregnanciesinteger0 – 20required
glucoseinteger1 – 600 mg/dLrequired
bloodpressureinteger1 – 200 mmHgrequired
skinthicknessinteger0 – 100 mmrequired
insulininteger0 – 900 IU/mLrequired
bmifloat1 – 80 kg/m²required
dpffloat0 – 3required
ageinteger1 – 120 yearsrequired
Example Request
{ "pregnancies": 2, "glucose": 180, "bloodpressure": 80, "skinthickness": 35, "insulin": 100, "bmi": 34.2, "dpf": 0.8, "age": 45 }
Response 200 — Success
{ "risk_percent": 78.5, "risk_category": "High", "prediction": 1, "confidence": 78.5, "message": "High diabetes risk detected (78.5%)" }
Response 422 — Validation Error
{ "error": "Validation failed", "details": ["Glucose must be between 1 and 600 mg/dL"] }
